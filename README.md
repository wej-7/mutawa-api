# 🌿 منصة متطوع الذكية

منصة إلكترونية متكاملة تربط المتطوعين بالجمعيات الخيرية بطريقة ذكية، بحيث يحصل كل متطوع على الفرصة المناسبة لمهاراته وموقعه.

## فكرة المشروع
كثير من الناس يريدون التطوع لكن لا يعرفون أين يبدؤون، وكثير من الجمعيات تحتاج متطوعين بمهارات محددة. منصة متطوع تحل هذه المشكلة بربط الطرفين تلقائياً.

## المميزات
- 🤖 ذكاء اصطناعي يقترح الفرص المناسبة لكل متطوع
- 🔐 نظام تسجيل دخول آمن
- 🏅 نقاط وشهادات رقمية بعد إتمام التطوع
- 📊 لوحة تحكم للجمعيات لإدارة الفرص والمتطوعين
- 🔒 حماية كاملة من الهجمات الإلكترونية

## التقنيات المستخدمة
- الواجهة: React.js
- السيرفر: Node.js + Express.js
- قاعدة البيانات: PostgreSQL
- المصادقة: JWT
- النشر: Render + Netlify
